Project overview
The Gym Management System is a software program designed to facilitate the management of client enrollment data. Users of this program can manage and add new clients' data.

Functionality:
1.	Adding client information
•	Navigate to the ‘Go to clients’ button.
•	Enter client information such as Name, Phone no., Email, Address.
•	Click the “save client’ button to add the client to the database.

2.	Managing the Instructors
•	Find the ‘Go to instructors’ button.
•	Enter the name and instructor Id and type.
•	Click on ‘Save instructor” button to save the information.

3.	Editing and deleting client details
•	Navigate to the ‘Go to clients’ button.
•	Find the client and then select the ‘edit’ button.
•	Update the information and select the ‘save client' button.
•	To delete the client information select the ‘delete’ button.

Problems solved
•	Manual Data Entry: Doing away with the necessity for manual data entry of client information saves time and decreases errors.
•	Centralized Database: Keeps client records in a centralized database, which facilitates information management and updating.
